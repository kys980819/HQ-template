# HQ — 나만의 AI 비서 뇌 🧠

> 폴더 하나에 **"뇌"를 만들고**, 옵시디언·클로드·안티그래비티(제미나이) 세 도구가
> 그 뇌를 함께 읽고 쓰게 만드는 개인용 세컨드 브레인 템플릿입니다.
> 데이터는 전부 내 컴퓨터의 `.md`(텍스트) 파일로 남아, AI 서비스를 갈아 끼워도 자산은 그대로입니다.

한국어 사용자를 위해 만들었고, 누구나 받아서 자기 것으로 쉽게 고칠 수 있습니다.

---

## 이게 뭔가요?

여러 AI를 쓰다 보면 대화가 흩어져 "내가 저번에 뭐라고 정리했더라?"가 반복됩니다.
HQ는 그 흩어짐을 막는 구조입니다. 핵심 흐름은 이렇습니다:

```
입력 → AI가 생각 → 뇌(옵시디언) 참고 → 필요하면 다른 AI에 지시
     → 결과물 → 다시 뇌에 기록  (기록이 쌓일수록 똑똑해지는 성장 루프)
```

- **뇌는 지도, 작업장은 땅.** 지식·기록은 `brain/`(옵시디언 볼트)에, 실제 코드·산출물은 `projects/`에 둡니다.
- **규칙은 한 곳에.** 모든 규칙은 `BRAIN.md` 한 파일에 있고, 각 AI용 설정 파일은 "이걸 먼저 읽어라"만 담은 얇은 문입니다.
- **데이터 소유권은 나에게.** 전부 로컬 텍스트 파일이라 특정 서비스에 묶이지 않습니다.

---

## 핵심 아이디어 3가지

1. **파일이 인터체인지** — AI끼리 결과를 주고받을 때 복붙하지 않고 볼트의 파일을 매개로 합니다. (제미나이가 조사해 저장 → 클로드가 읽어 가공)
2. **담당제** — 같은 일을 여러 AI에 던지지 않습니다. 제작은 클로드, 수집은 제미나이, 최종 검증은 코덱스(선택).
3. **저장 필터** — 아무거나 저장하면 뇌가 잡동사니가 됩니다. 5가지 조건 중 하나라도 맞아야 저장합니다.

---

## 폴더 한눈에 보기

```
HQ/
├── BRAIN.md            🧭 모든 규칙의 원본 (여기만 고치면 세 AI가 같이 따름)
├── CLAUDE.md / GEMINI.md / AGENTS.md   각 AI용 얇은 설정 (BRAIN.md를 불러옴)
├── inbox/              📥 정리 안 된 날것 자료를 던지는 곳 (AI가 흡수·가공)
├── projects/           🛠️ 실제 작업장 (코드·산출물)
└── brain/              🧠 옵시디언 볼트 = 뇌
    ├── index.md        볼트 전체 지도
    ├── log.md          AI 작업 기록
    ├── lessons.md      공용 교훈 (반복 실수 방지)
    ├── 00-빠른메모/     사람이 낮에 바로 적는 곳
    ├── 01-Wiki/         AI가 정리한 지식 (sources·concepts·decisions·errors·projects)
    ├── 10-취업/ 20-기업분석/ 30-학습/ 40-블로그/ 50-프로젝트/
    ├── 90-Daily/ 91-첨부/ 99-Archive/
    └── prompts/        자주 쓰는 프롬프트 원본
```

> `inbox/`와 `00-빠른메모/`는 이름이 비슷하지만 다릅니다 — `inbox`는 **정리할 날것 자료**, `00-빠른메모`는 **사람이 남기는 즉석 메모**입니다.

---

## 5분 빠른 시작

1. 이 폴더를 원하는 위치에 둡니다. (경로에 한글·공백이 없으면 더 안전)
2. **옵시디언**에서 `Open folder as vault`로 **`brain` 폴더**를 엽니다. (`HQ` 전체가 아니라 `brain`!)
3. 옵시디언 설정 2개: 첨부 기본 폴더 → `91-첨부`, Daily notes 폴더 → `90-Daily`.
4. **클로드코드**(또는 제미나이/안티그래비티)를 **`HQ` 폴더**에서 엽니다.
5. `BRAIN.md`의 맨 위 "사용자 프로필"을 내 정보로 채웁니다. 끝.

이제 자료를 `inbox/`에 던지고 `/ingest`라고 치면 AI가 뇌에 정리해줍니다.

---

## 명령어

클로드코드에서 `/`를 치면 나오는 명령들입니다.

| 명령 | 하는 일 |
|---|---|
| `/ingest` (흡수) | inbox의 새 자료를 읽어 위키로 정리 |
| `/query` (찾기) | "내가 저번에 뭐라 했더라?" — 뇌에서 찾아 요약 |
| `/lint` (점검) | 뇌 건강검진 (끊긴 링크·모순·민감정보 점검) |
| `/save` (저장) | 지금 대화 맥락을 규칙대로 저장 |
| `/아침브리핑` | 오늘 할 일·마감 임박을 데일리노트에 정리 |
| `/저녁마감` | 오늘 한 일 정리, 못 한 일 내일로 이월 |
| `/주간정리` | 빠른메모 분류·복습질문·건강검진·커밋 (주 1회) |
