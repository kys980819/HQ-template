# Lint Prompt (벤더 중립 원본)

```text
뇌(brain\)를 lint 해줘. 다음을 점검한다:

1. inbox 원본이 수정됐거나 위키 구역에 날것이 섞여 있지 않은가?
2. 중요한 위키 문서가 index.md 에 누락되지 않았는가?
3. 중요한 작업이 log.md 에 기록되지 않았는가?
4. 서로 모순되거나 낡은 정보(superseded)가 방치되지 않았는가?
5. 끊긴 [[위키링크]]가 없는가?
6. 저장 5필터를 통과 못 할 일회성 정보가 위키에 있지 않은가?
7. 민감정보(비밀번호·주민번호·금융·토큰)가 저장되지 않았는가?
8. lessons.md 에 3회 이상 반복된 교훈이 있으면 BRAIN.md 승격 후보로 표시

문제가 있으면 파일별 수정 제안만 정리한다 (직접 수정은 사용자 승인 후).
```
