# HQ 뇌 지도 (Index)

2026-07-08

이 문서는 볼트 전체의 지도다. AI는 중요한 위키 문서를 만들거나 갱신하면 **반드시 여기에 링크를 추가**한다.

## Start Here

- 공용 규칙 원본: `..\BRAIN.md` (볼트 밖, HQ 루트 — 어댑터 CLAUDE/GEMINI/AGENTS.md도 같은 곳)
- 처음이거나 구조가 궁금하면: HQ 루트의 `README.md`(요약)와 `사용설명서.md`(상세)
- [[log]] — 에이전트 작업 로그
- [[lessons]] — 공용 교훈 (모든 AI가 큰 작업 전에 읽기)

## 구역 안내

- `00-빠른메모\` — 사람이 낮에 떠오른 생각·링크를 바로 던지는 곳 (날것 자료는 볼트 밖 `inbox\`로 — 서로 다름). 주간정리 때 분류
- `01-Wiki\` — AI 전담 지식층: sources(자료 요약) / concepts(개념) / decisions(결정) / errors(실패 기록) / projects(프로젝트 맥락)
- `10-취업\` — [[지원현황]] · [[경험창고]] · 이력서\ · 자소서\(기업별 "회사명_직무\")
- `20-기업분석\` — 파일명 "회사명.md" (템플릿: [[템플릿-기업분석]])
- `30-학습\` — 보안\ · IT일반\ · 복습질문\
- `40-블로그\` — [[아이디어]] · 초안\ · 발행완료\
- `50-프로젝트\` — 프로젝트"에 대한" 노트 (코드는 볼트 밖 작업장)
- `90-Daily\` — 데일리노트 YYYY-MM-DD.md (템플릿: [[템플릿-데일리]])
- `91-첨부\` — 기록용 이미지 (옵시디언 첨부 기본 폴더)
- `99-Archive\` — 끝난 것들
- `prompts\` — 벤더 중립 프롬프트 원본
